----- ReadMe for DAB assignment #3 -----


GRUPPE 8

1. Åbn DAB_Assignment_3.snl fra vedlagte filer

2. Gå til filen "Program.cs", sørg for at connectionstring er sat til korrekt localhost

2. Kør' koden med CTRL+F5

3. Vent imens:
	- Database oprettes i mongoDB
	- Collections oprettes i databasen
	- DummyData indsættes

4. Følgende muligheder er tilgængelige i consollen:

	1. List all Users
	2: Select user by id
	3: Create Post
	4: Create Comment
	5: Create user
	Input Selection
	...

5. Start med at liste alle brugere med (1)

6. Efterfølgende kan (2) bruges 
	a) Se Feed for en given bruger (Kopier brugerID, for den bruger du ønsker at se feed)
	b) Se Wall for en given bruger (Kopier brugerID, for den bruger du ønsker at se wall hos, kopier derefter brugerID for den bruger, som besøger wall)

7. Navigér rundt som du/I vil


OBS: 'ecs' skal skrives med bogstaver (Det er altså ikke nok at trykke på esc knappen)
